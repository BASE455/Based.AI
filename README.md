# Based.AI Telegram Bot

AI Telegram bot with OCR + LLM support.

## Features

- Text processing with LLM
- Image text recognition (OCR)
- Multi-language support
- Telegram integration

## Tech Stack

- Python
- python-telegram-bot
- pytesseract
- requests

## Installation

1. Clone repository:
   git clone https://github.com/YOUR_USERNAME/Based.AI.git

2. Install dependencies:
   pip install -r requirements.txt

3. Create .env file and add your tokens

4. Run:
   python bot.py
